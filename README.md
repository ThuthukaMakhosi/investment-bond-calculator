# Investment & Bond Calculator

## Description
This is a simple Python program that allows users to calculate either investment returns or monthly home loan (bond) repayments. The program runs in any Python-supported code editor and uses console-based user input.

## Features
- Calculates simple or compound interest for investments  
- Calculates monthly repayments for home loans  
- Accepts user input via the terminal  
- Outputs results rounded to two decimal places for readability  

## Requirements
- Python 3.x  
- Any code editor (VS Code, PyCharm, IDLE, etc.)
